# Customer Segmentation using K-Means Clustering

## 📌 Overview
This project performs customer segmentation based on transaction data using clustering techniques.

## 🛠️ Technologies Used
- Python
- Scikit-learn
- Pandas

## ⚙️ Approach
- Cleaned and prepared numerical data
- Applied K-Means clustering
- Used Elbow Method and Silhouette Score for cluster selection

## 📊 Results
- Identified distinct customer groups
- Provided insights for business decision-making

## 🧠 Key Learning
Choosing the right number of clusters is crucial for meaningful segmentation.
